Ce qu'il faut tester

5 fonctionnalités à explorer

POST/agent/chat
💬 Chat Simple
Pose une question automobile générale à l'agent IA.
{ "message": "Quelle voiture familiale sous 30 000$ recommandes-tu ?" }
POST/agent/chat_web
🌐 Chat + Recherche Web
L'agent cherche sur internet en temps réel pour des offres actuelles.
{ "message": "Toyota Corolla 2022 moins de 20 000$ au Québec" }
POST/agent/analyze_url
🔗 Analyser une Annonce
Colle un vrai lien AutoTrader ou Kijiji pour une analyse complète.
{ "url": "https://www.autotrader.ca/a/toyota/corolla/..." }
POST/agent/compare_urls
⚖️ Comparer 2 Annonces
Donne 2 liens — l'agent choisit la meilleure option pour toi.
{ "url1": "autotrader.ca/...", "url2": "kijiji.ca/..." }
POST/agent/check_vin
🔍 Vérification VIN — Rapport gratuit
Entre le numéro VIN d'un véhicule (visible sur le tableau de bord côté conducteur) pour vérifier son historique, ses rappels de sécurité, et obtenir une recommandation d'achat.
{ "vin": "2T1BURHE0JC043821" }  ← Exemple VIN Toyota Corolla
